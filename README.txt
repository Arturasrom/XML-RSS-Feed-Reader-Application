Instruction:

In order to launch this application please follow these steps:

1. Open Windows Command Prompt(CMD) and go to project location by typing "cd" and your location path "C:\XML RSS READER\XML RSS READER".
2. To build application please type "mvn clean install". 
3. Go to location path "C:\XML RSS READER\XML RSS READER\target" and find .war file compiled for you.
4. Start Tomcat server and deploy .war file through Tomcat Web Application Manager.
5. Go to "http://localhost:8080/Rss_Feed_App/"
